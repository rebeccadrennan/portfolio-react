# Portfolio Website

React portfolio site for Rebecca Drennan, built with React, TypeScript, React Router, Bootstrap, and EmailJS.

## Features Implemented

- Routed sections for Home, About, and Contact with a persistent header and footer.
- Animated hero content with a typewriter effect, reveal-on-scroll transitions, and a custom cursor.
- Contact form with required fields, async submission, and success/error feedback through EmailJS.
- Responsive layout built with Bootstrap utilities so the site adapts cleanly across screen sizes.
- Accessibility-minded markup with semantic sections, alt text for imagery, and keyboard-friendly navigation.
- Theme toggle and navigation overlay for a more polished portfolio experience.

## Development

Install dependencies and run locally:

```bash
npm install
npm run dev
```

Run checks:

```bash
npm run build
npm run preview
```
